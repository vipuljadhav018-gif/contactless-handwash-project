# Contactless Handwash Project

## 📌 Overview
This project presents a **contactless handwash system** designed to improve hygiene by eliminating the need for physical contact. The system automatically dispenses soap or sanitizer when a user's hand is detected using a sensor.

## ⚙️ Features
- Touch-free operation
- Automatic soap/sanitizer dispensing
- Hygienic and user-friendly
- Low cost and easy to implement
- Reduces water and soap wastage

## 🧠 Working Principle
The system uses an **IR sensor** (or ultrasonic sensor) to detect the presence of a hand. When a hand is detected:
1. Sensor sends signal to controller
2. Controller activates relay
3. Pump dispenses soap for a fixed duration
4. System stops automatically

## 🔧 Components Used
- IR Sensor / Ultrasonic Sensor
